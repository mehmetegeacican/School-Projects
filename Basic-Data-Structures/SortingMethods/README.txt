******HOW TO WORK THE PROGRAM****************
1-DOWNLOAD THE FILE AND OPEN IT IN YOUR CMD (cd SortingMethods)
2- type--> g++ -o SortingMethods SortingMethods.cpp
3- type--> ./sortingMethods
***** IF YOU ARE USING LINUX*********
You can use make command to automatically compile
then type ./sortingMethods 

****THE PROGRAM******************
1- Enter an Array size
2- Decide Which Algorithm to use for sorting (1-Merge,2-Insertion,3-Quick,4-Hybrid)
3-[Optional] Print the sorted array if you want to,Type y for this when asked
4-Exit the Program or Sort again
